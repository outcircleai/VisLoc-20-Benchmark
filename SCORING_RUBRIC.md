# The VisLoc-20 Evaluation Protocol

## 🛑 GATE 1: The UI Physics Check (Pass/Fail)
* **[ Fail ] The Typographic Guillotine:** Was a word blindly sliced in half across a line break? 
* **[ Fail ] The Pacing Crash:** Did any subtitle card exceed 21.0 Characters Per Second? 
* **[ Fail ] The UI Collision:** Did text render over platform UI bounds? 
* **[ Fail ] The Cascading Edit Tax:** Does a human have to manually drag a timeline slider or hit 'Enter' to fix a line break? 
*(If the video fails Gate 1, it scores a 0/100).*

## 🧠 GATE 2: The Semantic Yield (Max 100 pts)
* **[ Base: +100 pts ]** Output survived the math and retained core context.
* **[ -50 pts ] Agency Lobotomy:** Flattened an active command into a passive fact.
* **[ -30 pts ] Context Hallucination:** Translated a proper noun or slang literally into gibberish.
* **[ -20 pts ] The Dangling Conjunction:** Stranded a weak conjunction at the end of a line.
* **[ +20 pts ] Bonus:** Successfully utilized a native cultural idiom to save characters.
